Q1. Difference Between Data Types in Python

Python provides different data types to store different kinds of values.

1. Integer (int)

Integer data type is used to store whole numbers without decimal points.

Example:

age = 20
2. Float (float)

Float data type is used to store decimal numbers.

Example:

price = 99.99
3. String (str)

String data type is used to store text enclosed in quotes.

Example:

name = "Shahid"
4. Boolean (bool)

Boolean data type stores only two values: True or False.

Example:

is_student = True
Data Type	Description	Example
int	Whole numbers	10, 25
float	Decimal numbers	10.5, 25.75
str	Text data	"Python"
bool	True/False values	True, False
Q4. Five Commonly Used String Methods in Python

String methods are functions used to perform operations on strings.

1. upper()

Converts all characters into uppercase.

name = "shahid"
print(name.upper())

Output:

SHAHID
2. lower()

Converts all characters into lowercase.

name = "SHAHID"
print(name.lower())

Output:

shahid
3. replace()

Replaces one word or character with another.

text = "Hello World"
print(text.replace("World", "Python"))

Output:

Hello Python
4. split()

Splits a string into a list.

text = "Apple Mango Banana"
print(text.split())

Output:

['Apple', 'Mango', 'Banana']
5. count()

Counts the occurrence of a character or word.

text = "banana"
print(text.count("a"))

Output:

3
Q7. What is Artificial Intelligence (AI)? Explain its Importance and Applications.

Artificial Intelligence (AI) is a branch of computer science that enables machines to think, learn, and make decisions like humans. AI systems can analyze data, recognize patterns, solve problems, and perform tasks automatically.

Importance of AI
Reduces human effort and saves time.
Improves accuracy and efficiency.
Helps in making better decisions.
Automates repetitive tasks.
Enhances user experience in technology.
Real-Life Applications of AI
Healthcare – Disease detection and medical diagnosis.
Education – Smart learning platforms and virtual tutors.
Banking – Fraud detection and customer support chatbots.
Transportation – Self-driving cars and route prediction.
Q8. Identify Whether the Following are Examples of AI and Explain Why
1. ChatGPT

Yes, it is an AI application.
It uses Artificial Intelligence and Natural Language Processing (NLP) to understand and generate human-like responses.

2. Google Maps Route Prediction

Yes, it is an AI application.
It analyzes traffic data and predicts the fastest route using machine learning algorithms.

3. Calculator

No, it is not an AI application.
A calculator only follows predefined mathematical rules and does not learn from data.

4. Netflix Recommendations

Yes, it is an AI application.
Netflix uses AI to analyze viewing history and recommend movies and shows according to user preferences.

5. Voice Assistants (Alexa/Siri)

Yes, they are AI applications.
They use speech recognition and AI algorithms to understand voice commands and perform tasks.
